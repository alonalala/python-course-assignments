Create a folder called day01

Add a file called day01/README.md and include all the information about the AIs you used.
Which AIs? Include links!
What were your prompts?
If you don't write your prompts in English that's fine. Include them as you wrote them.

The title of the issue should contain your full name and name of the assignment e.g. "Day1 by Foo Bar", in case your name is Foo Bar.
